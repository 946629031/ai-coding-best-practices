# Claude Code 的 Ralph Loop（拉尔夫循环）

- 给AI写好工作机会，每一步怎么测试验证，通过后就下一个任务
- 以此实现 7x24 小时 不停的代码开发

Claude Code 的 Ralph Loop（拉尔夫循环）是一种让 AI 编程助手实现自主、持续、不放弃地进行代码迭代的技术。它通过在任务完成后自动“喂”回相同的提示（Prompt）来阻止 Claude 过早退出，从而实现长期的自动化编程，常用于复杂任务执行、代码重构或单元测试完善。

### Ralph Loop 的关键特性和功能：

- **自主持续迭代（"永不放弃"）**：不同于传统的对话即停模式，Ralph Loop 允许 Claude 在编写代码、运行测试并发现错误后，自动继续修复，直到代码完全符合预设的“完成标准”。
- **Stop Hook 机制**：利用插件的停顿挂钩功能，在 Claude 想退出时强行将它“拦住”，使其重回工作状态。
- **自动化夜间任务**：允许开发者在下班前启动任务，让 Claude 自主完成项目框架迁移、依赖项升级或大型文档生成。
- **提高开发效能**：核心原理是将复杂工程拆解为稳定的回合制（任务票），本轮做不到就下一轮修，确保代码质量和可靠性。
